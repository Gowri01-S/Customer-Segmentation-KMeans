# Customer Segmentation using K-Means Clustering

## Objective
Group customers of a retail store based on their Annual Income and Spending Score using the K-Means Clustering algorithm.

## Dataset
Mall Customers Dataset

## Algorithm
- K-Means Clustering

## Tools Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Features Used
- Annual Income (k$)
- Spending Score (1-100)

## Outcome
Successfully segmented customers into 5 clusters using the Elbow Method and visualized the clusters.
